# jarvis-ai-assistant
Python-based AI voice assistant with speech recognition and desktop automation.

A Python-based AI voice assistant capable of:
- Speech recognition
- Opening applications
- Web searching
- Volume control
- Desktop automation

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- pyautogui

## Features
- Voice commands
- Open apps
- Search Google
- Control system volume
